Throughout the duration of my course I will submit 4 milestone projects. These will showcase what I have learned during each quarter stage of the course. My first two projects focused on front end. The first was purely HTML / CSS. The second then incorporated JaveScript and JQuery.

CSS Fundamentals: CSS Type and Class Selectors, CSS ID and Attribute Selectors,CSS DOM Selectors, CSS - Inline and Embedded, External CSS and precedence, CSS Rules, Comments and font-family, Working with Text, Font Parameters, Font Color, Font Position, Font Spacing, Letter Spacing, Font Style, The Box Model, Box Border and Margin, Background Appearance, Background Positioning, Background Source, Box Sizing, Layout, Visibility, Positioning, Z-index. As part of this module, I was also introduced to the Bootstrap framework.

JavaScript Fundamentals: Introduction, Commenting Your Code, Seeing What's Going On, Variables, Working With Numbers, Number Types, Number Methods and Objects, Strings, String Escaping, String Methods, Arrays, Array Methods, Array Dimensions and Concatenation, Decisions Decisions..., Operators, Type Conversion, Loops and Iteration, Break and Continue, Functions, Calling Functions, Keywords, Function Scope, Object-Oriented Programming, this and Object Values, JavaScript in The Browser, The Document Object Model. I was also introduced to the JQuery framework while working on this module.
